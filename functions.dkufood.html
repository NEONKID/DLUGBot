2.1.2. DKU Food System Module¶

functions.dkufood.requestFoodMenu(event)[source]¶

캠퍼스 내 식당 메뉴 정보를 가져옵니다.

(캠퍼스 내에 존재하지 않는 식당을 가져올 경우, 오류 발생)

식단 정보는 최근 7일 까지만 크롤링이 가능합니다

Parameters

event (JSON) – { location: { campus: 캠퍼스 이름, restaurant: 식당 이름 } }

Returns

요청 성공 여부와 메시지를 JSON 형태로 전달

{ result: { success: 요청 여부, message: 식단 메뉴 정보 메시지 } }
